# Lab Standards

This directory is populated by the spinup script (`friends-innovation-lab/playbook/operations/automation/spinup.sh`) when a new project is created. The script clones `friends-innovation-lab/lab-standards` and copies the relevant standards documents here.

If you're seeing this README in a spun-up project with no other files in this directory, the spinup script did not complete cleanly. Re-run it or contact the lab lead.

For the canonical standards source, see: https://github.com/friends-innovation-lab/lab-standards
